# 2026년 6월 10일 - SELECT (SQL)

## SUV 평균 대여 요금
car_type이 'SUV'인 차의 daily_fee 평균을 반올림하여 출력

### 배운 것
- `ROUND(AVG(컬럼))` - 평균을 반올림
- 컬럼 별칭 지정: `ROUND(AVG(DAILY_FEE)) AVERAGE_FEE`

## 아이스크림 판매량 정렬
총주문량 기준 내림차순, 같다면 출하번호 기준 오름차순 정렬

### 배운 것
- `ORDER BY 컬럼1 DESC, 컬럼2` - 다중 정렬 조건
- SELECT할 컬럼을 문제에서 정확히 캐치하기! (FLAVOR만 출력)

## 과일 기반 아이스크림 조회
총주문량 3000 초과 + 주성분이 과일인 아이스크림을 내림차순 조회

### 배운 것
- JOIN + WHERE 조건 복합 사용
- 문자열 비교 시 반드시 `' '` 안에 넣기
