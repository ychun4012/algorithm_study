## 2가 포함된 부분 배열
arr에서 2가 모두 포함된 가장 작은 연속 부분 배열 반환

### 풀이
- first: 처음 2의 인덱스 저장
- last: 이후 2가 나올 때마다 갱신 (마지막 2의 인덱스)
- first를 0이 아닌 -1로 초기화해야 인덱스 0과 구분 가능!

### 케이스 처리
- 2가 없는 경우 → [-1] 반환
- 2가 하나인 경우 → [2] 반환
- 2가 둘 이상인 경우 → first ~ last 구간 반환

### 배운 것
- 인덱스 초기값은 -1로 설정해야 0번 인덱스와 구분 가능
- ArrayList → int 배열 변환 stream 문장 이제 외워지는 중 😊
