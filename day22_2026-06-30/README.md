# 2026년 6월 30일 - DAY 11 리스트(배열)

## 알파벳 개수 세기
A-Z, a-z 각각의 개수를 길이 52 배열에 담기

### 풀이
ASCII 코드값으로 대문자/소문자 구분 후 index 계산

### 배운 것
- ASCII 코드: 대문자 A-Z는 65-90, 소문자 a-z는 97-122
- 대문자 index: `charAt(i) - 65` ('A' = 0번 index)
- 소문자 index: `charAt(i) - 71`
  - 대문자가 26개(0~25)를 차지하므로 소문자는 26부터 시작해야 함
  - 'a'(97) - 71 = 26 이 되도록 71을 뺌

## 1인 인덱스 찾기
idx보다 크거나 같으면서 값이 1인 가장 작은 인덱스 찾기

### 풀이
idx부터 순회하며 1을 만나면 바로 return, 없으면 기본값 -1 반환

### 배운 것
- 첫 번째로 만족하는 값을 찾을 때는 for문 안에서 바로 return하는 것이 효율적
  (불필요한 순회 없이 바로 종료)
