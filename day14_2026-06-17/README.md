# 2026년 6월 17일 - DAY 8 조건문, 문자열 (일일 도전 과제)

## 주사위 게임 2
네 주사위의 결과에 따라 다른 점수를 계산하는 문제

### 풀이 과정
1. `count[7]` 배열로 각 숫자(1~6)가 몇 번 나왔는지 카운트
2. count 배열에서 최댓값(max)을 구해 switch문으로 분기
3. p, q, r 변수로 각 경우의 숫자를 저장해 계산

### case별 처리
- `case 4` (전부 같음): count[i] == 4인 i → `1111 * p`
- `case 3` (3개 같음): count[i] == 3인 p, count[i] == 1인 q → `(10 * p + q)²`
- `case 2` (2개씩 or 2+1+1):
  - kinds == 2 (2+2): → `(p + q) * |p - q|`
  - kinds == 3 (2+1+1): → `q * r`
- `case 1` (전부 다름): 가장 작은 값 p → `p`

### 배운 것
- count 배열로 빈도수를 관리하는 패턴!
```java
  int[] count = new int[7];
  count[a]++; count[b]++; count[c]++; count[d]++;
```
- 경우의 수가 많을 때 switch문으로 분기 + kinds로 세부 분기
- 코드가 길어지더라도 논리를 명확히 나누는 것이 중요
