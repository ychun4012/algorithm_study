### 배운 것
- `for문`이 아닌 `while문`을 써야 하는 경우 주의!

## 배수 위치에 1 더하기
s~e 범위에서 인덱스 j가 k의 배수이면 arr[j]에 1 더하기

### 배운 것
- `arr[j]`의 값이 k의 배수인지가 아니라 **인덱스 j**가 k의 배수인지 확인!
  → 문제 해석 능력이 중요하다
- k가 0이면 배수를 구할 수 없으니 `break`로 바로 탈출
