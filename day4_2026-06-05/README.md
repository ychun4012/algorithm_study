## 곱과 합 비교

### 배운 것
- 곱하기의 초기값은 반드시 **1**로 설정하기! (0으로 두면 결과가 항상 0)

## 홀짝 이어붙이기
list를 돌며 짝수면 even에, 홀수면 odd에 string으로 추가 후 int로 변환해 합산.

### 배운 것
- `StringBuilder.toString()` - StringBuilder를 String으로 변환 가능
- 숫자가 들어올 때마다 10을 곱하면 형변환 없이 자릿수를 늘릴 수 있다!
```java
  even *= 10;
  even += num;
```
