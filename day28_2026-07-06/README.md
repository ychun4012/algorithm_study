# 2026년 7월 6일 - SELECT (SQL)

## 온라인/오프라인 판매 통합 조회
2022년 3월 온/오프라인 판매 데이터를 하나로 합쳐 조회

### 풀이
UNION ALL로 두 테이블을 아래로 쌓고 오프라인의 USER_ID는 NULL로 표시

### 배운 것
- `JOIN` - 두 테이블을 **옆으로** 붙이기
- `UNION` - 두 테이블을 **아래로** 쌓기 (중복 제거)
- `UNION ALL` - 두 테이블을 **아래로** 쌓기 (중복 포함)
- UNION 사용 시 두 SELECT의 컬럼 수와 타입이 같아야 함
- 없는 컬럼은 `NULL`로 채워서 컬럼 수 맞추기
