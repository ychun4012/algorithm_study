## 홀짝 합
if문으로 짝수와 홀수를 묻고 for문으로 각각 처리

### 배운 것
```java
for(int i = n; i >= 0; i -= 2)
    answer += (n % 2 == 0) ? i * i : i;
```
for문을 하나로 합치고 삼항연산자로 짝수 홀수를 다르게 처리하는 방법!
