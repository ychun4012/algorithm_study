## 정수 찾기 - 기초

### 문제
정수 리스트 `num_list`와 찾으려는 정수 `n`이 주어질 때, `num_list`안에 `n`이 있으면 1을 없으면 0을 return

### 풀이
list에 들어있는 값을 하나씩 돌며 n과 일치하는지 확인한 후 answer에 값을 부여하였다.

## 문자열로 변환 - 기초

### 문제
정수 `n`이 주어질 때, `n`을 문자열로 변환하여 return

### 풀이
도저히 모르겠어서 꼼수 방법으로 일단 답을 얻어냈다.
다른 사람들의 풀이를 보니 본건 있는데 제대로 사용하는 법을 모르고 있었단걸 알게 되었다.

### 배운 것
- `Integer.toString(n)` - Integer 클래스의 정적 메서드
- `String.valueOf(n)` - 특정 값을 지정한 타입으로 변환하거나 가져올 때 사용하는 정적 메서드

### 시도한 것들 (실패)
- `(String)n` → 실패
- `toString(n)` → 실패
- Integer로 변환 후 `(String)n` → 실패
