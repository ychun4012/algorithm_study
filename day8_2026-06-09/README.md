## LAB3, LAB5 - JOIN

### 배운 것
- `JOIN ... USING(컬럼)` - 같은 컬럼명으로 조인할 때 간단하게 사용
- `LEFT JOIN` - 왼쪽 테이블 기준, 조건 불일치해도 왼쪽 행 모두 출력
- `RIGHT JOIN` - 오른쪽 테이블 기준, 사용되지 않는 행도 포함
- `SELF JOIN` - 같은 테이블을 별칭으로 나눠 자기 자신과 조인
  - 직원-관리자 관계처럼 같은 테이블 내 계층 구조를 표현할 때 사용
- 3개 이상 테이블 조인 시 `JOIN`을 연속으로 사용

## LAB4 - SubQuery

### 배운 것
- 서브쿼리는 WHERE절 안에 SELECT문을 넣어 동적으로 조건 생성
- `IN (서브쿼리)` - 여러 값과 비교할 때 사용
- `> (서브쿼리)` - 단일 값과 비교할 때 사용
- 서브쿼리 안에서도 JOIN 사용 가능
- 여러 조건을 `AND`로 연결해 복합 서브쿼리 작성 가능
