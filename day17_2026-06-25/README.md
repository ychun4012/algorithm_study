# 2026년 6월 25일 - DAY 10 문자열 (일일 도전 과제)

## 구간 뒤집기
s~e 구간을 뒤집어 앞부분 + 뒤집은부분 + 뒷부분으로 조합

### 배운 것
- `StringBuilder.reverse()` - 문자열 뒤집기 메서드!
- StringBuilder/StringBuffer 주요 메서드 정리
  | 메서드 | 설명 |
  |--------|------|
  | `reverse()` | 뒤집기 |
  | `insert(i, s)` | i번 인덱스에 삽입 |
  | `delete(s, e)` | s~e 삭제 |
  | `replace(s, e, str)` | s~e를 str로 교체 |
  | `charAt(i)` | i번 문자 반환 |

## 세로 읽기
m글자씩 가로로 적었을 때 c번째 열 글자들 추출

### 배운 것
- c번째 열의 시작 인덱스는 `c - 1` (n번째 → 인덱스는 -1)
- 같은 열의 간격은 m이므로 `i += m`으로 순회

## 나머지로 필터링
인덱스를 q로 나눈 나머지가 r인 위치의 문자 추출

### 배운 것
- 문제를 그대로 코드로 옮기면 되는 문제
- `charAt(i)` - i번째 문자 반환
- `substring(i)` 는 i부터 끝까지 추출이므로 한 글자만 필요할 땐 `charAt()` 사용!
