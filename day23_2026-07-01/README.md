# 2026년 7월 1일 - 최소 직사각형 (공동과제)

## 최소 직사각형
모든 명함을 수납할 수 있는 가장 작은 지갑 크기 구하기

### 핵심 아이디어
명함을 돌릴 수 있으니 **긴 쪽은 가로, 짧은 쪽은 세로로 통일!**
그러면 가로 최댓값 × 세로 최댓값이 최소 지갑 크기가 된다.

### 풀이
- sizes[i][0] > sizes[i][1] 이면 그대로, 아니면 뒤집어서 getMax()에 전달
- getMax()에서 가로/세로 각각의 최댓값 갱신
- 최종적으로 garoMax * seroMax 반환

### 배운 것
- 방향을 통일하는 아이디어가 핵심!
  가로세로를 그대로 비교하면 답이 안 나오고
  긴 쪽 / 짧은 쪽으로 정규화하면 간단해짐
- 클래스 필드로 변수를 선언해 메서드 간 공유 가능
