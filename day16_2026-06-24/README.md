# 2026년 6월 24일 - DAY 9 문자열, DAY 10 문자 (일일 도전 과제)

## 접미사 확인
모든 접미사를 구해 is_suffix와 비교

### 배운 것
- `endsWith(suffix)` - 문자열이 특정 문자열로 끝나는지 확인하는 메서드!
- 접미사(suffix): 특정 인덱스부터 끝까지
- 접두사(prefix): 처음부터 특정 인덱스까지

## 앞의 n글자
substring(0, n)으로 앞의 n글자 추출

## 접두사 확인
접미사에서 배운 것을 바로 적용!

### 배운 것
- `startsWith(prefix)` - 문자열이 특정 문자열로 시작하는지 확인하는 메서드
