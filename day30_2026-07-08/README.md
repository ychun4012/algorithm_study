# 2026년 7월 8일 - 폰켓몬 (공동과제)

## 폰켓몬
N/2마리를 선택할 때 가장 많은 종류의 폰켓몬을 선택하는 경우의 수

### 핵심 아이디어
자연어로 논리를 먼저 생각하고 코드로 옮기니 훨씬 수월했다!
- 종류 수 = HashSet 크기 (중복 제거)
- 선택 가능 수 = N/2
- 답 = min(종류 수, N/2)

### 예시
- [3,3,3,2,2,4] → set: {3,2,4} (크기 3), max: 3 → 답: 3
- [3,3,3,2,2,2] → set: {3,2} (크기 2), max: 3 → 답: 2

### 배운 것
- `HashSet` - 중복 없이 저장, value가 필요 없을 때 HashMap 대신 사용
- `Math.min(a, b)` - 두 값 중 최솟값
- 코드 짜기 전에 자연어로 논리를 먼저 정리하는 습관이 중요!
